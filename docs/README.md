## 🔎 我們怎麼分析？

```
Amazon 評論（最新 50-200 則）
        ↓
    AI 萃取關鍵資訊
    ├── 產品聲明 vs 用戶體驗（有沒有吹牛？）
    ├── 負評分類（❌ 功能失效 / ⚠️ 副作用 / 📦 物流問題）
    ├── 問題發生頻率（多少 % 的人遇到）
    └── 問題嚴重程度（小困擾 vs 安全風險）
        ↓
    產出報告（每日更新）
```
