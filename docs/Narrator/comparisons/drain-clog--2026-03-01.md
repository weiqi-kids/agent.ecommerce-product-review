## 問題是什麼？

<p class="key-answer" data-question="排水管堵塞該怎麼處理？">堵塞類型決定最佳產品：毛髮堵塞選 Liquid-Plumr（84% 成功率、$7.99），日常預防維護選 Drano 80oz（$0.10/oz 最低單位成本），化糞池系統或油脂預防選 Green Gobbler 酵素式。但任何化學產品對嚴重堵塞的失敗率都在 16-34%，專業水電工首推機械通管器（通管蛇/活塞），化學通管劑應視為最後手段。</p>

排水管堵塞是家庭最常見的水電問題之一，症狀包括：
- 水流緩慢或完全無法排出
- 排水時發出咕嚕聲
- 水槽/浴缸積水
- 異味從排水孔散出
- 多個排水口同時出問題（可能是主管線堵塞，需立即找專業水電工）

這個問題不僅造成生活不便，若不處理還可能導致管線損壞或更嚴重的水管問題。

## 為什麼會有這個問題？

<p class="key-answer" data-question="排水管堵塞的常見原因是什麼？">毛髮是浴室堵塞的首要原因（人平均每天掉落 50-100 根），與油脂和皂垢結合後形成頑固堵塞。廚房則以油脂凝固累積為主因。「可沖式」濕紙巾是水電工公認的禁忌品。</p>

根據專業水電工的統計，排水管堵塞的常見原因包括：

| 原因 | 說明 | 常見位置 |
|------|------|----------|
| **毛髮** | #1 堵塞原因，與油脂、皂垢結合形成頑固堵塞。人平均每天掉落 50-100 根毛髮 | 浴室排水管 |
| **油脂** | 熱油脂倒入後遇冷凝固，持續累積 | 廚房水槽 |
| **皂垢** | 肥皂中的脂肪與水中礦物質結合形成硬質沉積 | 所有排水管 |
| **食物殘渣** | 馬鈴薯皮、蛋殼、咖啡渣等 | 廚房水槽 |
| **「可沖式」濕紙巾** | 儘管標示可沖，實際上會累積造成堵塞 | 馬桶、主管線 |
| **礦物質沉積** | 硬水區域的鈣/鎂沉積，需數年累積 | 老舊管線 |
| **樹根入侵** | 樹根從管線裂縫長入 | 地下污水管 |

## 解決方法有哪些？

<p class="key-answer" data-question="疏通排水管有哪些方法？">機械式（活塞、通管蛇）最安全有效，是專業水電工的首選。化學式（Drano、Liquid-Plumr）速度快但有管線腐蝕風險。酵素式（Green Gobbler）最環保但需 8-24 小時。專業水刀清洗（$350-1200）是嚴重堵塞的終極方案。</p>

| 方法類型 | 原理 | 代表產品 | 適合誰 |
|----------|------|----------|--------|
| **機械式（活塞、通管蛇）** | 物理方式推動或移除堵塞物 | 活塞（$5-30）、通管蛇（$5-100+） | 所有人，水電工首推方法 |
| **化學式（鹼性通管劑）** | 氫氧化鈉產生高溫化學反應溶解有機物 | Drano Max Gel、Liquid-Plumr、Zep Advanced | 輕度堵塞、不想動手的用戶 |
| **酵素式** | 天然酵素/細菌分解有機物，8-24 小時起效 | Green Gobbler、Bio-Clean | 環保考量、化糞池系統、預防性維護 |
| **天然 DIY** | 小蘇打+醋的化學反應 / 熱水+洗碗精 | 小蘇打、白醋 | 輕度堵塞、預防性維護 |
| **專業水刀清洗** | 高壓水柱（1500-4000 PSI）沖刷管壁 | 專業服務（$350-1200） | 嚴重/反覆堵塞、定期深層維護 |

### 專業水電工的建議順序

1. **優先嘗試**：活塞、通管蛇（最安全、最有效）
2. **次要選擇**：天然方法（熱水+洗碗精、小蘇打+醋）
3. **第三選擇**：酵素式清潔劑（較安全、需時較長）
4. **最後手段**：化學通管劑（快速但有風險）
