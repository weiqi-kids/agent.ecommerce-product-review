## 重新密封問題詳情

### 這為什麼是嚴重問題？

交換卡牌遊戲（TCG）產品的**核心價值在於隨機性**。買家付費購買的是：

1. **公平的抽卡機會**：未開封產品保證每張卡的機率相同
2. **投資潛力**：未開封產品可能含有高價值稀有卡
3. **收藏完整性**：密封產品本身具有收藏價值

當產品被「重新密封」時，這些價值全部喪失：

- **篩選風險**：高價值卡可能已被移除，買家只得到「殘餘」
- **假卡風險**：原裝卡可能被替換為仿製品
- **投資價值歸零**：即使卡片完整，「疑似重新密封」也使產品失去轉售價值

## 為什麼 Amazon 購買風險較高？

### TCG 產品供應鏈特性

```
製造商 → 經銷商 → Amazon 倉庫 → 第三方賣家混入 → 消費者
                          ↑
                    風險進入點：
                    - 退貨產品重新上架
                    - 第三方賣家混合
                    - 倉庫內部人員篡改
```

### Amazon 平台特有風險

| 風險因素 | 說明 |
|---------|------|
| **混合庫存 (Commingled Inventory)** | 不同賣家的相同產品可能存放在一起，難以追溯來源 |
| **退貨政策** | TCG 產品退貨後可能被重新上架，即使已被開封 |
| **第三方賣家** | 即使標示「Ships from Amazon」，產品可能來自第三方 |
| **驗證困難** | 買家收貨前無法檢查密封完整性 |

### 卡片真偽辨識

#### 燈光測試

- **真卡**：對著明亮光線完全不透明，看不到背面
- **假卡**：透明，可看到背面圖案

#### 重音符號檢查

- **真卡**：「Pokemon」中的 "e" 有重音符號（e）
- **假卡**：缺少重音符號（明顯假冒跡象）

#### 卡片層結構（撕裂測試，僅限不要的卡）

- **真卡**：撕裂後露出中間的灰色箔片層
- **假卡**：便宜紙板製成，無夾層結構
