# 胃酸逆流 (Acid Reflux / GERD) 解決指南

## 問題是什麼？

胃酸逆流（Gastroesophageal Reflux Disease, GERD）是指胃酸與胃內容物逆流至食道，造成不適症狀的慢性消化系統疾病。

**主要症狀**：
- 胃灼熱（Heartburn）：胸口灼熱感，常在飯後或躺下時加劇
- 逆流感：酸性或苦味液體上湧至喉嚨或口腔
- 吞嚥困難、慢性咳嗽、聲音沙啞、喉嚨異物感

**影響範圍**：
- 全球約 10-20% 人口受影響
- 未治療可能導致 Barrett's 食道（癌前病變）、食道狹窄、食道腺癌等嚴重併發症
- 嚴重影響睡眠品質與生活品質

---

## 為什麼會有這個問題？

### 核心機制：下食道括約肌（LES）功能不良

下食道括約肌是食道與胃之間的「閥門」，正常情況下應在吞嚥時開啟、其餘時間關閉。當 LES 在不當時機放鬆或壓力不足（<8 mmHg）時，胃酸就會逆流。

> "Acid reflux doesn't have anything to do with the acid in the stomach, as everybody has the same amount of acid in their stomach - it's a mechanical problem with the valve" - Cleveland Clinic

### 主要風險因素

| 因素 | 影響 |
|------|------|
| **體重過重** | 肥胖者 GERD 風險增加 2-3 倍，腹部脂肪增加胃內壓力 |
| **橫膈膜疝氣** | 95% GERD 患者伴隨此問題，影響 LES 功能 |
| **飲食習慣** | 油炸食物、辛辣、柑橘、番茄、巧克力、咖啡因、酒精 |
| **生活型態** | 吸菸、壓力、睡前進食、久坐 |
| **藥物** | NSAIDs、某些降血壓藥可能惡化症狀 |
| **懷孕** | 2/3 孕婦經歷 GERD（荷爾蒙與腹壓變化） |

## 解決方法有哪些？

| 方法類型 | 原理 | 代表產品 | 適合誰 |
|---------|------|----------|--------|
| **制酸劑 (Antacids)** | 直接中和胃酸，立即緩解 | Tums, Rolaids, Mylanta | 偶發輕度症狀者 |
| **H2 阻斷劑** | 阻斷組織胺受體，減少胃酸分泌 | Famotidine (Pepcid) | 中度症狀、夜間症狀為主 |
| **PPI (質子幫浦抑制劑)** | 不可逆抑制胃酸分泌最終步驟，24小時持效 | Omeprazole, Nexium | 頻繁症狀（每週2+次）、食道炎 |
| **Vonoprazan (P-CAB)** | 競爭性結合，不需酸性環境活化 | Vonoprazan | PPI 難治型患者 |
| **手術治療** | 修復 LES 結構或增強功能 | Nissen Fundoplication, LINX | 長期 PPI 無效、大型疝氣 |
| **生活方式調整** | 減少誘發因素、改善 LES 功能 | 減重、抬高床頭、飲食控制 | 所有患者（基礎治療） |

### 它如何解決問題？

Omeprazole 屬於質子幫浦抑制劑（PPI），是目前治療 GERD 最有效的藥物類別：

**作用機制**：
1. 作為前驅藥物進入體內
2. 在胃壁細胞的酸性環境中活化為 sulfenamide
3. 與 H+/K+ ATPase（質子幫浦）形成不可逆共價鍵
4. 阻斷胃酸分泌的最終步驟（三種刺激途徑全部阻斷）

**臨床效果**（基於醫學研究）：
- GERD 症狀緩解率：85-95%
- 相比 H2 阻斷劑：症狀緩解提升 35%、黏膜癒合提升 50%
- 嚴重食道炎治癒率 >85%（vs. H2 阻斷劑 40-60%）
- 減少胃酸分泌達 99%

**Delayed Release（腸溶劑型）重要性**：
- Omeprazole 對胃酸敏感，腸溶包衣保護藥物通過胃部
- 在小腸（pH >5）釋放吸收，再經血液到達胃壁細胞作用

## 長期使用注意事項

基於 ACG/AGA 臨床指引與醫學研究：

### 短期使用（<3 個月）
- 風險極低，適合急性症狀控制
- 無需特殊監測

### 長期使用（>3 個月）建議
- 每 6-12 個月評估是否仍需使用
- 年度檢測：血清鎂、維生素 B12
- 骨折高危險群：補充鈣（1200mg/day）+ 維生素 D（800-1000 IU/day）
- 有腎病家族史：更密切腎功能監測

### 停藥策略
1. 症狀控制後嘗試逐步減量（20mg → 10mg → 隔日）
2. 或改用 H2 阻斷劑維持
3. 若出現反彈，短期恢復後再次嘗試
4. 持續強化生活方式調整
