# 商品跨平台對應表（Product Registry）

本文件維護所有追蹤商品的跨平台對應關係。

## 使用方式

- **新增商品**：使用 `track product {URL}` 指令，或手動編輯本文件
- **product_id**：以 ASIN 或 UPC/EAN 為主鍵
- **跨平台關聯**：同一 product_id 可對應多個平台/店家
