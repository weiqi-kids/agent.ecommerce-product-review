## 問題定義

### 核心問題
嬰兒（0-12 個月）需要適齡的感官刺激來促進大腦發育，但家長面臨：
- 不知道哪些玩具適合特定月齡
- 擔心材質安全（BPA、鄰苯二甲酸酯、鉛）
- 市面產品品質參差不齊
- 過度刺激 vs 適度刺激的平衡

### 問題成因
1. **發育需求**：嬰兒大腦在出生後第一年發育最快，需要視覺、觸覺、聽覺刺激
2. **安全顧慮**：80% 玩具在中國製造，品質管控風險
3. **資訊不足**：家長不清楚各月齡適合的感官類型
4. **市場混亂**：大量聲稱「感官」的玩具實際效果不明

## 月齡分階指南

### 0-3 個月
| 發展重點 | 適合玩具類型 | 原因 |
|---------|------------|------|
| 視覺追蹤 | 高對比黑白玩具 | 新生兒只能看到黑白灰 |
| 聽覺刺激 | 輕柔搖鈴、音樂盒 | 發展聲音定位能力 |
| 觸覺探索 | 柔軟安撫巾、lovey | 安全感建立 |

### 3-6 個月
| 發展重點 | 適合玩具類型 | 原因 |
|---------|------------|------|
| 抓握能力 | 紋理球、軟質積木 | 訓練手部肌肉 |
| 因果關係 | 皺紋書、按壓玩具 | 理解動作產生結果 |
| 趴臥時間 | 趴臥鏡、活動墊 | 強化頸部和核心肌群 |

### 6-12 個月
| 發展重點 | 適合玩具類型 | 原因 |
|---------|------------|------|
| 物體恆存 | 躲貓貓玩具、蓋布遊戲 | 理解物體消失後仍存在 |
| 精細動作 | 堆疊杯、形狀分類 | 手眼協調 |
| 探索慾望 | 感官球、水墊 | 滿足好奇心 |

## 安全標準與認證

### 美國 CPSC 規範
| 項目 | 要求 |
|------|------|
| 鄰苯二甲酸酯 | 所有兒童玩具不得超過 0.1%（1000 ppm） |
| 鉛含量 | 油漆不得超過 90 ppm |
| 窒息風險 | 3 歲以下玩具零件不得小於 1.25" x 2.25" |
| 第三方測試 | 12 歲以下兒童產品須第三方認證 |

### 安全認證標章
| 認證 | 意義 |
|------|------|
| ASTM F963-23 | 美國玩具安全標準 |
| CPSIA | 消費品安全改進法合規 |
| CE | 歐盟安全標準 |
| GOTS | 有機紡織品全球標準 |
| Oeko-Tex | 無有害物質認證 |

### 家長自檢方法
- **衛生紙捲測試**：零件能穿過衛生紙捲即有窒息風險
- **氣味測試**：強烈化學味道代表可能含有害物質
- **標籤檢查**：確認 BPA-free、Phthalate-free 標示

## 特殊需求考量

### 自閉症/發育遲緩
- **鎮靜瓶**：視覺追蹤、情緒調節（Jellystone Calm Down Bottle 獲高評價）
- **加重玩具**：提供本體感覺輸入
- **簡單設計**：避免過度刺激

### 蒙特梭利 vs 傳統玩具
| 面向 | 蒙特梭利 | 傳統 |
|------|---------|------|
| 材質 | 天然木材、布料 | 塑膠為主 |
| 顏色 | 自然色調 | 鮮豔色彩 |
| 功能 | 單一目的、開放式 | 多功能、電子 |
| 價位 | 較高 | 較低 |

## 購買建議

### 價格區間

| 區間 | 價格 | 適合場景 |
|------|------|---------|
| 經濟 | <$15 | 試用、派對禮物 |
| 主流 | $15-$30 | 日常使用、生日禮物 |
| 高端 | $30-$60 | 有機/手工品牌 |
| 奢華 | >$60 | 收藏級、傳家寶 |

### 採購優先順序
1. **安全第一**：確認無小零件、無毒材質
2. **月齡適配**：選擇適合發展階段的玩具
3. **品牌信譽**：優先選擇有認證的品牌
4. **多感官**：結合視覺、觸覺、聽覺刺激
5. **開放式玩法**：避免功能過於單一
