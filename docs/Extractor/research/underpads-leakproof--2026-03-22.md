## 1. 問題定義

### 核心問題
失禁（urinary incontinence）患者在睡眠或臥床期間，尿液溢出導致床單、床墊及皮膚長時間接觸濕氣，引發：
- 皮膚潮濕破損（moisture-associated skin damage, MASD）
- 壓瘡風險升高（pressure ulcer）
- 異味（odor）
- 頻繁更換床單（caregiver burden）

### 目標使用者
- 老年失禁患者（佔市場需求約 61%）
- 醫院/護理機構住院患者（佔 48%）
- 居家護理者（佔 37%）
- 術後恢復者
- 嬰幼兒尿床

## 2. 問題成因

| 成因 | 說明 |
|------|------|
| 膀胱肌肉衰弱 | 老化、產後、神經系統疾病 |
| 認知退化 | 阿茲海默症、失智症患者無法自主如廁 |
| 行動不便 | 骨折、中風後遺症，無法即時移動到廁所 |
| 夜間多尿 | 腎功能衰退、利尿藥物影響 |
| 壓力性失禁 | 咳嗽、打噴嚏時漏尿 |

全球失禁患者超過 **4.2 億人**，市場需求持續成長。

## 3. 解決方案比較

| 方法 | 優點 | 缺點 |
|------|------|------|
| **一次性失禁墊（underpad）** | 易換、高吸收、成本可控 | 環境廢棄物多 |
| **可水洗失禁墊** | 環保、長期較省錢 | 需洗滌、重度失禁保護力有限 |
| **成人紙尿褲** | 隨身攜帶、不限臥床 | 穿脫不便、熱悶 |
| **導尿管** | 徹底防漏 | 感染風險（CAUTI），僅限醫療情境 |
| **床墊防水保護套** | 長期保護床墊 | 不吸收液體，需搭配其他產品 |

**一次性失禁墊（chux/underpad）是居家護理和醫療機構最主流的床面保護方案**，每個長照患者每日使用 3–6 片。

## 4. 產品技術原理

### 五層結構（標準配置）
1. **頂層 topsheet**：不織布（polypropylene/polyester），快速導液、柔軟親膚
2. **絨毛漿層 fluff pulp**：木漿纖維，初步吸收液體
3. **SAP 層（超吸收聚合物）**：polyacrylate，吸收自重 30 倍液體，形成凝膠鎖水
4. **組織層 tissue**：均勻分散液體，防止集中
5. **防水底層 backsheet**：PE/PP 薄膜，阻止液體滲透床面

### SAP 技術差異
- **標準型**（fluff only）：45–60 gram，適合輕度失禁
- **聚合物型**（fluff + polymer）：吸收力提升 2–3 倍
- **進階聚合物型**（heavy polymer/SAP beads）：吸收 10 倍以上（如 Medline Ultrasorbs 宣稱）

### 透氣設計
部分高端產品採用空氣透過型底層（air-permeable backsheet），可減少熱積聚，降低皮膚刺激風險，特別適合低氣壓氣墊床（low air-loss bed）患者。

## 5. 使用者體驗關鍵面向

根據多方評測與用戶評論彙整：

### 正向體驗
- 高吸收力（尤其夜間不需頻繁更換）
- 頂層快乾，減少皮膚潮濕感
- 不易撕裂（翻身/移動患者時重要）
- 防滑底面（SlipResist）

### 常見抱怨
- **位移/滑動**（slipping/bunching）：無防滑底面的產品容易移位
- **吸收力不達標**（leakage through）：低克重產品在重度失禁中效果差
- **品質下降**：部分品牌近年出現「越來越薄」的用戶投訴
- **價格偏高**：高吸收力產品（Medline Ultrasorbs、NorthShore MagicSorb）單片成本顯著高於基本款
