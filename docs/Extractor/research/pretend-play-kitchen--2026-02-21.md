## 問題背景

### 核心需求

幼兒（2-6 歲）透過家家酒廚房玩具發展多種關鍵能力：

1. **假裝遊戲能力** - 模仿成人活動，建立對世界的理解
2. **社交技能** - 學習分享、輪流、合作
3. **語言發展** - 擴展食物、烹飪、用餐相關詞彙
4. **精細動作** - 切割、攪拌、擺盤等動作訓練
5. **認知發展** - 計劃、排序、問題解決

### 研究支持

根據 Child Mind Institute 研究：
- 假裝遊戲支持「情緒知識和情緒調節控制」發展
- 促進「心智理論」(Theory of Mind) - 理解他人可能有不同想法的能力
- 增強「短期記憶和靈活思考」
- 2-5 歲是假裝遊戲的「高峰期」

## 木質 vs 塑膠比較

| 特點 | 木質（KidKraft等） | 塑膠（Step2等） |
|------|------------------|----------------|
| 耐用性 | 高（可翻新） | 中（刮傷無法修復） |
| 美觀度 | 高（融入家居） | 中等 |
| 清潔 | 需小心 | 容易擦拭 |
| 組裝時間 | 長（30分鐘-數小時） | 短 |
| 價格 | 較高 | 較低 |
| 戶外使用 | 不建議 | 可以 |
| 穩定性 | 高（較重） | 中（可能傾倒） |

## 發展效益

### 語言發展

根據語言治療師研究，廚房玩具有助於：

| 技能 | 練習方式 |
|------|---------|
| 名詞詞彙 | 食物名稱、廚具名稱 |
| 動詞 | cut, stir, cook, eat, pour |
| 介係詞 | in, on, under, inside |
| 描述詞 | hot, cold, yummy, ready |
| 對話能力 | 角色扮演對話 |

### 精細動作

| 活動 | 發展目標 |
|------|---------|
| 切割食物 | 手眼協調、握力 |
| 攪拌 | 手腕旋轉 |
| 倒水/倒茶 | 平衡、控制 |
| 擺盤 | 空間規劃 |

### 社交情緒

| 技能 | 練習方式 |

## 購買建議

### 依年齡選擇

| 年齡 | 建議類型 | 原因 |
|------|---------|------|
| 1-2 歲 | 玩具食物籃 + 購物車 | 簡單探索 |
| 2-3 歲 | 小型廚房 + 基本食物 | 開始假裝遊戲 |
| 3-5 歲 | 中型廚房 + 多樣配件 | 假裝遊戲高峰 |
| 5+ 歲 | 大型廚房 | 複雜角色扮演 |

### 依空間選擇

| 空間 | 建議產品 |
|------|---------|
| 小公寓 | Hape 緊湊型（22"x13"x28"）|
| 中等房間 | KidKraft Vintage |
| 大遊戲室 | Step2 Grand Walk-In |

### 依預算選擇

| 預算 | 建議 |
|------|------|
| <$50 | 玩具食物套組 + 基本用具 |
| $50-100 | deAO 或 Chad Valley 小型廚房 |
| $100-200 | KidKraft 或 ROBUD 中型廚房 |
| $200+ | Lalo 或 Step2 大型廚房 |

---

## 安全注意事項

1. **年齡標示** - 大多數標示 3 歲以上（小零件窒息風險）
2. **材質認證** - 選擇無毒漆料（水性漆）
3. **穩定性** - 確保不易傾倒
4. **圓角設計** - 避免尖銳邊緣
