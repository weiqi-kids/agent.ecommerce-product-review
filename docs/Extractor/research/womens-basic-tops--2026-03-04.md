# 女性基本上衣品質問題研究

## 研究摘要

女性基本上衣（basic tops / t-shirts / rib knit tops）的核心品質問題包含：洗後縮水（shrinkage）、起毛球（pilling）、褪色（fading）、尺碼失真、面料薄透、以及快時尚品質與定價不相符的性價比問題。代表產品 ANRABESS Rib Knit T-Shirt 是 Amazon 熱銷的羅紋針織基本款，主打修身設計和多色選擇。市場正面臨快時尚（廉價但低質）與 D2C 品牌（合理定價但高質）的兩極競爭。

## 問題成因

### 縮水（Shrinkage）
- **根本原因**：天然纖維（棉、羊毛）在紡織過程中被拉伸，遇熱水或高溫烘乾時回縮至原始長度
- **觸發因素**：
  - 高溫洗滌（> 30°C）
  - 熱烘乾（機器烘乾）
  - 高棉含量（100% 棉縮水率最高）
  - 未經預縮處理的廉價面料

### 起毛球（Pilling）
- **根本原因**：纖維末端斷裂後糾結成球，短纖維混紡最嚴重
- **觸發因素**：
  - 棉/聚酯混紡（聚酯毛球不易脫落，堆積更明顯）
  - 腋下/胸側等高摩擦區域
  - 機器洗滌的強力攪拌
  - 低 GSM 面料（< 180 GSM）

### 褪色（Fading）
- **根本原因**：低品質染料滲透深度不足，或對熱/光敏感
- **觸發因素**：
  - 高溫洗滌分解染料分子
  - 直接日曬（UV 分解）
  - 洗滌劑過強

### 面料薄透
- **根本原因**：追求低成本降低 GSM，面料密度不足
- **觸發因素**：
  - GSM < 160 的羅紋針織
  - 過高彈性比例（Elastane > 10%）造成結構疏鬆
  - Rayon 等下垂感強的材質在淺色款透光

### 尺碼失真
- **根本原因**：快時尚品牌缺乏標準化尺碼系統，或針對特定體型設計
- **觸發因素**：
  - 中國品牌尺碼普遍偏小（需加購一至兩號）
  - 修身版型（slim fit）誤差更明顯

### 品質指標

- **GSM（克/平方米）**：180-280 GSM 為優質面料；< 160 GSM 偏薄
- **捻紗數（Yarn Count）**：數字越高，纖維越細，越柔軟
- **GOTS 認證**：確保有機製程無有害化學品
- **OEKO-TEX Standard 100**：確認不含有害物質

## 副作用與風險

- **快時尚環境衝擊**：過度生產快時尚造成大量紡織廢棄物
- **染料化學品**：低品質染料含有甲醛、偶氮染料等有害物質
- **Rayon 製程**：黏膠法製造 Rayon 需大量化學溶劑
- **尺碼歧視**：部分品牌尺碼範圍有限，對大尺碼消費者不友善

## 專家意見

- **紡織專家**：GSM 220-260 是最平衡的面料重量，兼顧質感與透氣
- **時尚編輯**：Supima 棉或長絨棉製成的上衣，5 年後仍保持品質
- **永續顧問**：有機棉天然纖維強度較高，不需化學處理增加強度
- **消費者顧問**：寧可購買一件 $40 好品質，勝過四件各 $10 廉價品

## 市場趨勢
