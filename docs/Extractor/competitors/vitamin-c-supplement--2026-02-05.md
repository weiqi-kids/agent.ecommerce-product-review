## 競品詳情

### 1. Emergen-C（維生素 C 粉末）
- 品牌：Emergen-C
- ASIN：N/A
- 推薦理由：
  - 市場領先的維生素 C 補充品
  - 每份 1000mg 維生素 C、B 群維生素、電解質
  - Immune+ 版本添加維生素 D、鋅
  - 快速溶解，口味美味清爽
  - 多種口味選擇（Super Orange 評價最高）
- 來源：深度研究報告「免疫系統支援與維生素補充」第 3.1 節
- 價格：每份約 $0.60（10 包 $6）

### 產品優點
- 快速溶解，提供能量和免疫力提升感受
- 秋季家中必備，感冒初期服用有感（使用者評價）
- 含 B 群維生素和電解質，複合配方

### 產品缺點
- 糖含量過高（標準配方每包 6g 添加糖，佔每日攝取量 12%）
- 價格持續上漲（長期用戶抱怨）
- 含人工甜味劑和添加劑

### 科學有效性分析
- GoodRx 評估：Emergen-C 未在臨床研究中證明有效
- 醫學評論：1000mg/天維生素 C 使成人感冒持續時間減少 8%
- Illuminate Labs 結論：因有效維生素 C 劑量可能有效預防和減少疾病持續時間，但因含糖和添加劑被認為不健康
- 鋅含量過低（2mg vs. Airborne 8mg），無法提供研究觀察到的治療益處（需每日 75mg+）

### 法律爭議（2023）
1. **虛假廣告案（2023-01）**：聲稱含「天然水果香料」，實際含合成香料 DL 蘋果酸
2. **維生素 C 含量不足案**：軟糖產品實際含量僅標示的 55%（409mg vs 750mg）
3. **TheraFlu 組合包誤導案（2023-06）**：誤導消費者認為可緩解感冒和流感症狀

### 維生素 C 科學證據
- 每日補充 1000mg+ 可減少成人感冒持續時間 8%、兒童 14%
- 嚴重症狀減輕 15%（95% CI 9-21%）
- **重要限制**：無法減少健康人群感冒發生率；症狀出現後服用無效
- 特殊人群（馬拉松跑者、滑雪者、士兵）感冒發生率減少約 50%

### 作用機制
1. 上皮屏障功能：保護皮膚免受環境氧化壓力
2. 吞噬細胞增強：增強趨化、吞噬、ROS 生成、微生物殺滅
3. 淋巴球功能：增強 B 細胞和 T 細胞分化與增殖
4. 表觀遺傳調控：加速 B 細胞分化為抗體產生漿細胞
5. 抗發炎效應：降低促發炎細胞因子（TNF、IL-6、IL-1β）

### 安全性與副作用
- 常見副作用（>2000mg/天）：胃部不適、嘔吐、大便鬆散、腹瀉、噁心
- 腎結石：高劑量可能導致腎結石，特別是腎臟疾病患者
- 嚴重健康問題：
  - 溶血性貧血（G6PDH 缺乏症患者）
  - 鐵過載（遺傳性血色素沉著症或地中海貧血患者）
  - 血糖讀數錯誤（糖尿病患者）
  - 懷孕期間過度補充可能影響寶寶

### 專家建議
- 幾乎不可能從食物中攝取過多維生素 C，副作用主要來自補充品
- 大多數醫生不建議維生素 C 補充品，除非醫生建議
- 對於能夠獲得正常健康飲食且沒有臨床維生素缺乏的個體，維生素補充無證實的臨床益處
