### 安全標準與認證
**UL 1449標準 (第5版, 2025年10月)**:
- SPD安全和性能標準
- Type 1: 服務變壓器與主配電斷路器之間
- Type 2: 全屋突波保護器
- Type 3: 終端使用SPD (距配電盤≥10米)
- Type 4: Type 5組件與斷路器組成
- Type 5: 離散組件 (如MOV)

**箝位電壓 (VPR)**:
- 330V優於500V
- 測定VPR突波電流從500A提高至3000A

**必備認證**:
- UL認證 (必須選購)
- 表示經過安全測試並符合嚴格標準

### 火災危害統計
- **美國消防署 (USFA)**: 電器故障每年~51,000起住宅火災, ~500人死亡, ~1,400人受傷, 損失~13億美元
- **美國電氣安全基金會 (ESFI)**: 延長線和電源插座每年>3,300起火災, 50人死亡, 270人受傷

### 關鍵安全守則
1. **禁止菊鏈連接** (daisy-chaining): 違反OSHA和NEC
2. **僅使用有內建斷路器**的產品
3. **定期檢查**: 磨損、焦痕、插座鬆動
4. **高瓦數家電禁用延長線**: 電暖器、微波爐、冰箱直插牆面
5. **遠離水源或熱源**
6. **3-5年更換一次**: MOV劣化隨時間累積
