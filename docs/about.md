# 關於我們

## 買前必看 — 電商商品評論智慧分析系統

我們是一個專注於電商評論數據分析的 AI 團隊，運用機器學習技術分析數萬則真實用戶評論，為消費者提供客觀的購買參考。

---

## 我們的使命

在資訊爆炸的時代，消費者面對海量商品評論時往往難以判斷：

- 哪些評論是真實的？
- 產品真正的優缺點是什麼？
- 這個產品適合我嗎？

**買前必看** 透過 AI 技術，從數千則評論中萃取關鍵洞察，幫助你做出明智的購買決策。

---

## 我們的方法

### 資料來源

| 來源 | 說明 |
|------|------|
| Amazon 評論 | 美國 Amazon 真實購買者評論 |
| 專業評測 | 知名媒體與評測網站 |
| 社群討論 | Reddit、論壇等用戶討論 |

### 分析技術

1. **評論真實性檢測** — 識別異常評論模式
2. **情感分析** — 量化正負評價比例
3. **面向萃取** — 識別用戶關注的產品面向
4. **問題聚類** — 歸納共同問題與痛點
5. **競品比較** — 跨產品橫向分析

### 報告類型

| 類型 | 說明 |
|------|------|
| ✅ 推薦報告 | 產品表現優異，值得購買 |
| ⚖️ 比較報告 | 多產品橫向比較分析 |
| ⚠️ 警告報告 | 發現安全或品質問題 |
| 📊 痛點報告 | 市場共同問題分析 |
| 🚨 假貨報告 | 假貨風險警示 |
