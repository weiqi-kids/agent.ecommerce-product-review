## 各類別痛點摘要

### 維他命 C 補充品
{: .text-yellow-300 }

**主要問題**：
- Emergen-C 固化問題率高達 28%
- 2024 訴訟進行中（維生素C含量不實、天然風味標示）
- 整體市場品質參差不齊

### 延長線
{: .text-red-300 }

**主要問題**：
- 多品牌（HANYCONY、ANNQUAN、HEZI）有火災報告
- 突波保護失效案例
- ANNQUAN、HEZI 已召回

### 紓壓玩具
{: .text-yellow-300 }

**主要問題**：
- 化學氣味問題普遍
- 材質安全疑慮
- 品質控制不穩定
